# Bathymetry of Lagoa da Conceição: Interactive Leaflet Map &amp; GPS

🔗 Confira o mapa aqui <https://mauritia-flexuosa.github.io/mapa_batimetrico/>

📄 Os dados batimétricos utilizados são provenientes de <https://geoportal.pmf.sc.gov.br/downloads/camadas-em-sig-do-mapa>

#### Este projeto apresenta um mapa interativo da batimetria da Lagoa da Conceição (Florianópolis, Brasil), desenvolvido em linguagem R utilizando o framework Leaflet.

#### O mapa permite a visualização das profundidades sobrepostas a imagens de satélite e inclui recursos de geolocalização em tempo real.

## 🚀 Funcionalidades

#### Visualização de Profundidade: Gradiente de cores representando as variações batimétricas.
#### Geolocalização (GPS): Botão interativo para localizar o usuário no mapa (útil para navegação em campo).
#### Controle de Camadas: Alternância entre imagem de satélite (Esri World Imagery) e mapa de ruas (OpenStreetMap).
#### Pop-ups Interativos: Clique nos polígonos para visualizar a profundidade exata em metros.
#### Interface Otimizada: Botões e legendas ampliados para facilitar o uso em dispositivos móveis (tablets e smartphones).

## 🛠️ Tecnologias Utilizadas.
<p>O projeto foi construído utilizando as seguintes bibliotecas do R:</p>

#### sf: Manipulação de dados geográficos (Shapefiles).

#### leaflet: Criação de mapas interativos.

#### leaflet.extras: Funcionalidades adicionais como o plugin de GPS.

#### htmltools: Customização de estilos CSS e pop-ups.
